Images

From https://developer.mozilla.org/en-US/docs/Learn/HTML/Multimedia_and_embedding/Images_in_HTML:

  1. In your HTML from assignment one, wrap each <img> inside a <figure> container.

  2. Add a <figcaption> container and a caption inside the <figure> container after the <img>.

    Note that the alt text and the caption should not say the same thing.

The result should look something like this:

<figure>

<img src="images/cobalt-hair.jpg" alt="I got a bright blue mohawk haircut for church choir">

<figcaption>I was trying to get out of having to participate in choir.</figcaption>

</figure>

CSS

From https://developer.mozilla.org/en-US/docs/Learn/CSS/Styling_text/Fundamentals:

Make these changes to your linked CSS:

  1. Add a default font stack (a font-family value consisting of multiple font names separated by commas) for the whole site, by assigning the rule to the html tag.

  2. Set a default base font-size of 10px for the html tag.

    • 10px is too small to be useful, but we are using it as the base because it's really easy to do math on.

    • For all other font sizes, we will use relative ems (rem), which are computed from the base font-size.

    • Why don't we just set all font sizes in pixel sizes?
      By using rem, we can just change the base font-size and automatically change all font sizes in a website.

  3. Set the font-size for p and li to 1.2em and see how it looks. (Probably too small). Try 1.4em. Keep adjusting till it looks the way you want.

  4. Set the font-size for h1 to 2.3em and see how it looks. (Probably too small). Try 2.4em. Keep adjusting till it looks the way you want.

  5. Set the font-size for h2 to 2em and see how it looks. (Probably too small). Try 2.1em. Keep adjusting till it looks the way you want.

  6. Set the line-height on li

Do more if you like. At a minimum, the result should look something like this:

html { font-size: 10px; }

h1 { font-size: 2.6rem; }

p, li { font-size: 1.4rem; font-family: Helvetica, Arial, sans-serif; }

li { line-height: 1.5; }

From https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps/Getting_started:

Change how hyperlinks display, by making these changes to your linked CSS:

You can use the site at https://www.w3schools.com/colors/colors_picker.asp to pick some colors.

  1. Change the color for an unvisited link (a:link).

  2. Change the color for a visited link.

  3. Change the CSS so when you mouse over a link, the background color and the link color both change, and the underline disappears.

  4. Change the CSS so when you click a link, the background color and the link color reverse, and the underline disappears. (a:active)

  5. For the header and main elements, add 2.5rem (which adds up to 25px) padding so the text isn't sitting directly against the side of the page.

Do more if you like. At a minimum, the CSS should now look something like this:

html {
font-size: 10px;
font-size: 1.4rem;
font-family: Helvetica, Arial, sans-serif;
}

header, main { padding: 2.5rem; }

h1 { font-size: 2.6rem; }

li { line-height: 1.5; }

a:link { color: #e60000; }

a:visited { color: #990073; }

a:hover { color: #cc3300; background-color: #ebebe0; text-decoration: none; }

a:active { color: #ebebe0; background-color: #cc3300; text-decoration: none; }
