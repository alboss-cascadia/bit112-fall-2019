GitHub Desktop

These instructions are stolen with permission from Brad Ives, who documented them way better than I could have.

You can find Github Desktop at https://desktop.github.com/

Once you have it downloaded, log in. After that look in Github's settings and select Visual Studio Code as the external editor. It looks like the attached image.

Make sure your remote repository is set up. To do that go to Github.com, log in and then take note of the address bar:

Now go into Github Desktop, click repository>repository settings and paste the copied URL into the spot that says primary remote repository (origin). Click save.

Now push the "fetch origin" button in the top left of Github Desktop. after it synchs, you should be able to make changes in VS code and push them from there.

When you make an edit in VS code you then save (ctrl+s) and then stage the edit by pushing the plus button, add a couple words for description and then click the check mark.

Then go back into Github Desktop and either press ctrl+P or push the blue "push origin" button on the upper right. this will publish your changes directly to your site. Also your Github repository will match what you have on your PC. hooray!
