Project plan

Title

Whale Tales

Goal

To inspire people with the beauty of Humpback Whales and provide information about how they can help this endangered species.

Target Audience

The site is intended to reach people who are interested in knowing more about Humpback Whales, including animal lovers, whale lovers, marine biologists, and environmentalists.

Main Features

Web site with links to information, stories and pictures of humpback whales. A gallery of images will provide a sense of the grandeur of humpback whales in their natural habitat. A sign-up form for an email action-alert newsletter is also provided.

Personality

Bold, majestic, appealing, simple, clear, urgent.

The site will help people feel more passionate about helping these majestic animals.

Limiting Factors

Website will have an image gallery with large pictures so internet capabilities should be fairly current. Audience should have some experience with Internet but does not need to be at the intermediate or expert level.

Development guide

Site structure

  • Root Directory
    • images directory – contains jpgs and gifs
    • styles directory – contains all of the style sheets
    • docs directory – contains information documents as pdfs
    • index.html – home page for site, site purpose and overview
    • facts.html – information about humpback whales with graphics and links
    • support.html – information on saving whales with images, graphics and links
    • watching.html – gallery of images with information on whale watching and links
    • about.html – board membership, history, organization structure
    • contact.html - email and mailing address

Styles guide

Background

Background is light yellow

{background-color: #FFFFCC}

Default font

Text is dark blue, 14 pixels

p { font-size: 14px; color: #003366;}

Header

The header section is 235px tall, positioned along the top of each page with the logo centered in the frame. The background is Sea Blue.

header {
height: 235px;
background-color: #336699;
background-image: url("/images/whale.png");
background-repeat: no-repeat;
background-position: center;
background-size: cover;
}

Navigation

The main navigation is 120px tall, positioned just below the header with a background color of Light Yellow and Dark Blue font color.

nav { height: 120px; background-color: #ffffcc; color: #003366; }

Images

Images will be over light blue background with a grey border around the container

figure { background-color: 6699CC; border: 2px solid #333333; }

img { padding: 1em; }

Links

Links will be burgundy. On hover, they will have a light blue background and a blue border.

a { color: #993333 }

a:hover { background-color: #6699CC; border 2px dotted #336699; }
