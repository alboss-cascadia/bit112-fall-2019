Here's a handy tip for those of you who are using Visual Studio Code as your editor. Let's say you want to change several consecutive instances of "> to " target="_blank"> on a certain page. Highlight the first instance of it, then hold down the command key (Mac) or control key (Windows) and press the letter D. Each time you press the D it'll highlight another instance. Once you've highlighted what you wanted, type the replacement and it'll type it in all the highlighted places.

CodePen is a really cool tool for experimenting with stuff. It lets you type html, type css, and preview, all at once. It doesn't fit in with what we're doing since the goal is complete pages of HTML that are put into Github, but it's awesome for figuring stuff out.
